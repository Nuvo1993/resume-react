# Component Map

This document lists the main UI components in the project and their responsibilities.

- Header
  - Path: `src/components/Header/Header.js`
  - Description: Top header with name/title, may include a hero area.

- AppBar / NavBar
  - Path: `src/components/AppBar/NavBar.js`
  - Description: Navigation bar used for in-page navigation between resume sections.

- AboutMe
  - Path: `src/components/AboutMe/AboutMe.js`
  - Description: Short biography / summary section.

- Projects
  - Path: `src/components/Projects/Projects.js`
  - Description: A list or grid of project cards with links and descriptions.

- Resume
  - Path: `src/components/Resume/Resume.js`
  - Description: Resume layout and a download link/button to get a PDF.

- Contact
  - Path: `src/components/Contact/Contact.js`
  - Description: Contact details and optional contact form.

If you add new components, please update this file with the path and a 1-2 line summary.
